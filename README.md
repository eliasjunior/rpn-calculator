# RPN Calculator

## Getting Started

### Prerequisites

* Java 11

* Maven Version 3.6.3

### Installation

1. Clone the rpn calculator
   ```sh
   git clone https://github.com/eliasjunior/rpn-calculator.git
   ```
2. Install Maven dependencies packages, run the command bellow.
   ```sh
   mvn package
   ```
